Boilerexams Application Project

This project is a React based statistics dashborad that analyses local user submission data. It displays metrics like accuracy, total attempts, average number of attempts etc.

The data displayed by the dashboard:
  1. Total questions and attempts
  2. Accuracy and first-try accuracy metrics
  3. Average attempts per question
  4. Attempts distribution per question
